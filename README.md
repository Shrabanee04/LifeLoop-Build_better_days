# 🚀 LifeLoop — Build Better Days

LifeLoop is an AI-powered personal productivity and life management platform designed to help users organize their daily lives, improve productivity, and build better habits through a single intelligent ecosystem.

## ✨ Features

- 🔐 Authentication & User Management
- 📊 Smart Dashboard
- 📅 Smart Calendar & Scheduling
- ✅ Task & Productivity Management
- 🧠 AI Journal System
- 😊 Mood Tracking
- 💰 Finance Tracker
- 🤖 AI Personal Assistant
- 🧠 AI Memory Recall
- 🌦️ Weather-Based Suggestions

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Framer Motion

### Backend
- Spring Boot
- JWT Authentication
- REST APIs

### Database
- MySQL

### AI Layer
- Python FastAPI
- OpenAI API / Gemini API

## 🎯 Objective

LifeLoop aims to provide a centralized platform where users can manage tasks, schedules, journals, moods, and finances while leveraging AI-powered insights and recommendations to improve daily productivity and personal growth.

## 📌 Key Highlights

- Personalized productivity insights
- Smart scheduling assistance
- AI-powered journaling and mood analysis
- Finance management and budgeting
- Weather-aware recommendations
- Secure and responsive user experience
